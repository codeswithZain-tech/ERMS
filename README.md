# Examination Result Management System (C++)

## Project Overview

The **Examination Result Management System** is a C++ console-based application designed to manage student academic records efficiently.
It supports adding, updating, deleting, searching, and analyzing student results using **Data Structures and Object-Oriented Programming (OOP)** concepts.

---

## Features

### 🔹 Student Management

* Add new student
* Remove student
* Search student by ID
* Update student marks

### 🔹 Result Processing

* Automatic GPA calculation
* Grade assignment (A, B, C, D, F)
* Pass/Fail status

### 🔹 Data Structures Used

* Linked List
* Binary Search Tree (BST)
* AVL Tree
* Heap
* Stack (Undo/Redo functionality)
* Queue

### 🔹 Advanced Functionalities

* Undo / Redo system
* Action history tracking
* Merit list generation
* Top/Bottom K students
* Statistics (average, highest, lowest, grade distribution)
* Export data to CSV

---
